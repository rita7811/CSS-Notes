CSS Transition

display ===>
visibility ===>

Explanation:

  • display property (from block to none): Transition Not Supported (added 1s delay does nothing)
  • visibility property (from visible to hidden): Instant transition after 1s delay
  • opacity property (from 1 to 0): Smooth transition with 0.5s duration after 1s delay
